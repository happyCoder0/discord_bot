Вот список моих команд:
/help - выводит список команд
/abw example - добавляет слово example в список запрещенных. За употребление запрещенных слов я выдаю пользователям предупреждения.Три предупреждения и бан на 1 день (Требуется разрешение на бан участников).
/dbw example - удаляет слово example из списка запрещенных.
/gbw - выводит все запрещенные слова.
/request url - позволяет сделать запрос к сайту по адресу url.
/mywarns - выводит количество ваших предупреждений.
/aic example - добавляет чат с id example в список игнорируемых.
/dic example - удаляет чат с id example из списка игнорируемых.
/gic - выводит id всех игнорируемых чатов
